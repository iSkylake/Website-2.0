About

Hey! I'm a computer enthusiast and programmer who is always looking to learn more and try new tehcnologies. I got my first interest with computers when I was 4 years old just clicking icons and writing words in the Notepad. Later when I tried my first computer video game and used the Internet for the first time with Netscape, I discovered a new whole world.

Back in Highschool, I designed my first website using HTML and CSS. I remember adding a lot of cool images and GIFs and messing with the style, at the end it looked like some kind of tumblr. In College is where I touched programming for the first time learning different languages like C, C++, Java, VB, C#, etc. and solving complex engineering problems with data structures. In my Robotic class playing with Arduino was really fascinating, programming a microcontroller to do things in the real world.

After I finished College, I came to San Francisco Bay Area, a whole new world was presented to me and I got more excited about my software engineering career. I learned full stack web development, I can do the front-end and the backend using the MERN stack. I got interest in mobile development, Machine Learning and VR/AR. Some of the platforms I enjoy using are TwitchTV, Reddit, GitHub and Steam.
